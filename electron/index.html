기업용: 개인 계정 대신 전용 계정을 사용하세요. 크롤링 시작 시 로그인 상태를 확인하고, 미로그인이면 이 계정으로 로그인합니다.

크롬 내 크롬 프로필로 크롤링 (사용 불가)
크롬은 자동화로 로그인된 프로필을 열면 "본인 인증"을 요구하며 프로필을 잠그고, 닫을 때 구글 계정을 로그아웃시킵니다(우회 불가한 크롬 보안). 그래서 이 방식은 비활성화되었고, 항상 전용 프로필로 크롤링합니다. 네이버는 계정만 입력하면 자동 로그인됩니다.
네이버 확인 전
쿠팡 확인 전
쿠팡은 봇차단이 강해, 열리는 크롬 창에서 직접 잠깐 둘러보면 통과됩니다.
[쿠팡 준비] 클릭 → 열린 창에서 상품 몇 개 클릭·스크롤(원하면 로그인) → "준비 완료" 뜨면 끝. 이후 쿠팡 크롤이 됩니다.
세션 쿠키 붙여넣기 (로그인·2차인증 건너뛰기)
본인 크롬에서 coupang.com 로그인 → Cookie-Editor(cookie-editor.com) 확장 → 하단 Export(JSON, 클립보드 복사) → 여기 붙여넣기.
전체를 Export해도 쿠팡·네이버만 자동 저장. ⚠ hotcleaner "Cookie Manager"는 암호화 export라 안 됩니다 — Cookie-Editor를 쓰세요. 차단된 회사망은 프록시 필요.

프록시 레지덴셜 프록시로 크롤 (쿠팡 IP차단 우회)
회사 공인 IP가 쿠팡(Akamai)에 차단된 상태입니다. 주거용(residential) 프록시로 깨끗한 IP를 태우면 통과됩니다.
서버 입력 → 저장 → 프록시 테스트로 IP·쿠팡 접속 확인 → 토글 ON 후 크롤.
