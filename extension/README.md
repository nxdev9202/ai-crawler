# NX 쿠키 전송 확장 (Chrome)

쿠팡·네이버 로그인 세션을 **버튼 한 번**으로 NX AI 크롤러에 전송하는 크롬 확장입니다.
기존 방식(Cookie-Editor로 Export → 복사 → 앱에 붙여넣기, 3단계)을 **클릭 1번**으로 줄입니다.

> **왜 필요한가:** 크롬은 보안(App-Bound Encryption)으로 프로그램이 쿠키를 자동으로 읽는 걸
> 막습니다. 하지만 크롬 확장은 `chrome.cookies` 권한으로 **httpOnly 쿠키(로그인·봇우회 세션)까지**
> 정식으로 읽을 수 있어, 이 확장이 그 세션을 NX 앱으로 안전하게 넘겨줍니다.

---

## 1. 설치 (최초 1회)

1. 크롬 주소창에 **`chrome://extensions`** 입력 → 이동
2. 우측 상단 **"개발자 모드"** 토글 **켜기**
3. 좌측 상단 **"압축해제된 확장 프로그램을 로드합니다"** 클릭
4. 이 **`extension` 폴더**를 선택
   - (NX 앱 설치 경로 안, 또는 받으신 `extension` 폴더)
5. 확장 목록에 **"NX 쿠키 전송"** (NX 골드 아이콘)이 뜨면 성공
6. (선택) 주소창 옆 퍼즐 아이콘 → **NX 쿠키 전송** 옆 **압정(📌)** 클릭 → 툴바에 고정

---

## 2. 사용법 (매번)

1. **NX 앱을 먼저 켜둡니다** (백엔드가 실행 중이어야 함)
2. 크롬에서 **coupang.com 에 로그인** (네이버도 쓸 거면 naver.com 도 로그인)
3. 툴바의 **NX 쿠키 전송 아이콘** 클릭
4. 팝업에서 원하는 버튼 클릭:
   - **🛒 쿠팡 세션 전송** — 쿠팡 쿠키만
   - **🟢 네이버 세션 전송** — 네이버 쿠키만
   - **📦 쿠팡 + 네이버 모두 전송** — 둘 다
5. **`✓ 전송 완료! 저장됨: 쿠팡 N`** 이 뜨면 끝
6. NX 앱에서 크롤하면 그 세션으로 진행됩니다

> 그게 전부입니다. Export도, 복사·붙여넣기도 필요 없습니다.

---

## 3. 언제 다시 눌러야 하나

- 쿠키는 **로그아웃하거나 만료되기 전까지 유효**합니다 (보통 며칠~몇 주)
- 크롤 시 쿠팡이 다시 막히면(로그인 풀림) → 사이트에 재로그인 후 버튼 한 번 더

---

## 4. 문제 해결

| 증상 | 원인 / 해결 |
|---|---|
| **"전송 실패 … NX 앱이 켜져 있는지"** | NX 앱(백엔드)이 꺼져 있음 → 앱을 켠 뒤 다시 |
| **"쿠키가 없습니다"** | 그 사이트에 로그인이 안 됨 → coupang.com 로그인 후 다시 |
| **"쿠팡/네이버 쿠키 없음"** | 다른 탭이었거나 로그인 안 됨. 쿠팡 로그인 상태 확인 |
| 포트가 다르다는 안내 | 팝업 하단 "NX 앱 포트"는 기본 **8756** — 바꾸지 마세요 |

---

## 5. 개인정보 / 보안

- 이 확장은 **쿠팡·네이버 쿠키만** 읽어 **내 PC의 NX 앱(127.0.0.1:8756)** 으로만 보냅니다.
- 외부 서버로 전송하지 않습니다. 구글·다른 사이트 쿠키는 읽지도, 보내지도 않습니다.
  (백엔드도 쿠팡·네이버 외 쿠키는 버립니다.)
- 코드는 `popup.js` 전체가 공개되어 있어 검증 가능합니다.

---

## 6. 사내 배포 팁

- `extension` 폴더를 사내 공유(사내망/USB)로 나눠주고, 각자 위 **1. 설치**만 하면 됩니다.
- 개발자 모드 로드가 번거로우면, Chrome 정책(GPO)의 `ExtensionInstallForcelist` +
  사내 웹에 올린 `.crx`로 강제 배포도 가능합니다(선택).
