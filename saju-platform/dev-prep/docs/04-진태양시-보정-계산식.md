# 04. 진태양시(眞太陽時) 보정 계산식

> 시주(時柱)는 "태양이 실제로 어디에 있는가"를 기준으로 세우므로,
> 시계 시각(표준시)을 출생지의 실제 태양시로 환산해야 한다. 보정은 두 단계다.

## 1단계: 경도 보정 (지방평균태양시, LMT)

표준시는 기준 자오선(현재 동경 135°)의 시각이다. 출생지 경도가 다르면 그 차이만큼 보정한다.

```
경도 보정(분) = (출생지 경도 − 기준 자오선 경도) × 4분/도
지방평균태양시 = 표준시 + 경도 보정
```

- 경도 1° = 4분 (지구 자전 15°/시간)
- **기준 자오선은 시대별로 다름** (`korea_timezone_history.json` 참조): 현재는 135°, 1954~1961년은 127.5°.

### 주요 도시 경도 보정표 (기준 자오선 135° 기준)

| 도시 | 경도 | 보정(분) |
|---|---|---|
| 서울 | 126.98°E | **−32.1** |
| 인천 | 126.71°E | −33.2 |
| 대전 | 127.38°E | −30.5 |
| 광주 | 126.85°E | −32.6 |
| 대구 | 128.60°E | −25.6 |
| 부산 | 129.08°E | −23.7 |
| 울산 | 129.31°E | −22.8 |
| 제주 | 126.53°E | −33.9 |
| 강릉 | 128.90°E | −24.4 |
| 독도 | 131.87°E | −12.5 |

→ 서울 출생자는 시계보다 태양이 약 32분 늦다. "서울 기준 −30분" 관행은 이 값의 근사.

## 2단계: 균시차(Equation of Time) 보정 — 선택

태양의 실제 움직임은 계절에 따라 평균태양시 대비 ±16분가량 빠르거나 늦다 (지구 공전궤도 이심률 + 자전축 기울기).

```
진태양시 = 지방평균태양시 + 균시차(EoT)
```

- 균시차 범위: 약 −14분(2월 중순) ~ +16분(11월 초)
- 근사식 (분 단위, 연중일 N 기준):

```
B = 2π × (N − 81) / 365          # N = 1월 1일부터의 일수
EoT = 9.87·sin(2B) − 7.53·cos(B) − 1.5·sin(B)   (분)
```

- 더 정확하게는 pyephem/천문 계산으로 구할 수 있으나, 근사식으로도 ±수십 초 이내.

## 권장 구현 방침

| 옵션 | 내용 | 권장도 |
|---|---|---|
| A. 보정 없음 | 표준시 그대로 | 비권장 (경계 출생자 시주 오류) |
| B. 경도 보정만 | 표준시 + 경도차×4분 | **기본값 권장** — 시중 만세력 대부분 이 방식 ("−30분") |
| C. 경도 + 균시차 | 완전한 진태양시 | 고급 옵션으로 제공 (역술가마다 견해 갈림) |

- **사용자 입력은 "출생 도시" 선택**으로 받고, 도시→경도 매핑 테이블로 처리 (위 표 + 시군구 확장).
- 출생지를 모르면 서울(−32분)을 기본값으로.
- 시주 경계(예: 자시 23:00~01:00 등 홀수시 경계)를 넘나드는지가 관건이므로, 보정 후 시각으로 시지(時支)를 판정한다.

## 시대별 종합 예시

```
예) 1955-08-08 16:20 서울 출생 (시계 시각)
 [1] 서머타임 기간(1955-05-05~09-09) → −1h → 15:20
 [2] 당시 표준시 UTC+8:30 (기준 자오선 127.5°)
 [3] 경도 보정: (126.98 − 127.5) × 4 = −2.1분 → 15:17.9  ← 이 시대는 자오선이 한반도 관통이라 보정이 작다
 [4] (옵션) 균시차 8월 8일 ≈ −5.6분 → 15:12
 → 미시(未時, 13:00~15:00)와 신시(申時, 15:00~17:00) 경계 근처 판정에 사용

예) 1990-03-15 15:05 서울 출생
 [1] 서머타임 아님
 [2] UTC+9 (135°)
 [3] 경도 보정: −32.1분 → 14:32.9 → 미시(未時)로 확정 (보정 전엔 신시)
```
