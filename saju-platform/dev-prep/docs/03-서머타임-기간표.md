## 적용 파이프라인 (시각 정규화 순서)

```
출생시각(시계) 
  → [1] 서머타임 기간이면 −1시간
  → [2] 해당 시대 표준시 오프셋 확인 (UTC+9 또는 UTC+8:30)
  → [3] 진태양시 보정 (→ 04 문서)
  → 보정된 시각으로 시주/일주 경계 판정
```

⚠️ **주의: 1955~1960년은 서머타임과 UTC+8:30 시대가 중첩**된다. 두 보정을 모두 적용해야 한다.
예: 1955-08-08 16:20 출생(시계) → 서머타임 −1h → 15:20 (UTC+8:30 표준시) → UTC 06:50.

⚠️ **참고 논점**: 블로그 원문에도 언급되듯 "서머타임 시간을 사주에 적용하느냐"는 역술가마다 견해가 갈린다.
기술적으로는 **적용(−1h 환원)이 기본값**이고, 설정 옵션으로 끄고 켤 수 있게 만드는 것이 시중 만세력의 표준 관행이다.
